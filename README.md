# 🚀 LifeOS

> A modern personal productivity dashboard built with React.

LifeOS is an all-in-one productivity platform designed to help users organize their daily life through task management, note-taking, expense tracking, profile customization, and theme settings.

---

## ✨ Features

### ✅ Todo Manager

* Add, delete and manage tasks
* Mark tasks as completed
* Persistent storage using Local Storage

### 📋 Notes Manager

* Create and organize notes
* View recent notes directly from the dashboard
* Delete notes easily

### 💰 Expense Tracker

* Add expenses with categories
* Track spending
* View expense records

### 👤 Profile Management

* Personal profile settings
* Name, college, branch and bio support
* Persistent profile storage

### ⚙️ Settings

* Dark Mode support
* Dashboard customization
* Reset individual sections
* Reset the entire application

### 🏠 Smart Dashboard

* Overview of todos, notes and expenses
* Recent activity section
* Quick action buttons
